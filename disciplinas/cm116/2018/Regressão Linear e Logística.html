  • Queremos encontrar um modelo $h_{\theta}(x)$ tal que $h_{\theta}(x^i)$ aproxime bem $y^i$;
  • Função de perda $\ell$: $\ell(h_{\theta}(x^i), y^i)$;
  • Minimizamos a média das perdas $$ L(\theta) = \sum_{i = 1}^m \ell(h_{\theta}(x^i), y^i); $$

Modelo linear com perda quadrática - QUADRADOS MÍNIMOS¶

  • $ h_{\theta}(x) = \theta_1 \phi_1(x) + \theta_2 \phi_2(x) + \dots + \theta_n \phi_n(x) = \theta^T\phi(x), $ cada $\phi_j:\mathbb{R}^p\to\mathbb{R}$, e $\phi:\mathbb{R}^p \to \mathbb{R}^n$.
  • $\ell(\hat{y}, y) = \frac{1}{2}(\hat{y} - y)^2$.
  • $$ \min L(\theta) = \frac{1}{2}\sum_{i=1}^m \big(\theta^T\phi(x^i) - y^i\big)^2. $$

Este problema em particular pode ser escrito como $$ \min L(\theta) = \frac{1}{2}\Vert X\theta - y\Vert^2, $$ onde $$ X = \left[\begin{array}{ccc} \phi_1(x^1) & \dots & \phi_n(x^1) \\ \phi_1(x^2) & \dots & \phi_n(x^2) \\ \vdots & & \vdots \\ \phi_1(x^m) & \dots & \phi_n(x^m) \end{array}\right]. $$

Esse problema é convexo, e se $X$ tem colunas linearmente independentes, é estritamente convexo, e tem solução única. As soluções do problema de minimização são soluções do sistema linear $$ X^TX \theta = X^Ty. $$

  • Existem diversas maneiras de se resolver esse sistema. Se $X$ tem poucas colunas, podemos realmente montar o sistema e resolvê-lo.
  • Podemos usar fatorações;
  • Podemos usar o método dos gradientes conjugados.
  • No problema de minimização, podemos utilizar métodos de otimização para funções quadráticas.
  • Em particular, podemos usar o método de gradiente estocástico.

# Solução da sabatina AKA Cauchy para quadráticas
function cauchy(X, y; ϵ = 1e-6)
    m, n = size(X)
    θ = zeros(n)
    d = X' * y
    iter = 0
    while norm(d) > ϵ
        Xd = X * d
        α = dot(d, d) / dot(Xd, Xd)
        θ .+= α * d
        d = X' * (y - X * θ)
        iter += 1
    end
    return θ, norm(y - X * θ)^2, iter
end

using ForwardDiff

function gradiente(L, θ₀; η = 0.01, ϵ = 1e-4, max_iter = 10000)
    θ = copy(θ₀)
    ∇L(θ) = ForwardDiff.gradient(L, θ)
    ∇Lθ = ∇L(θ)
    iter = 0
    while !(norm(∇Lθ) <= ϵ || iter > max_iter)
        t = 1.0
        slope = -dot(∇Lθ, ∇Lθ)
        while !(L(θ - t * ∇Lθ) < L(θ) + 1e-2 * t * slope)
            t *= 0.5
            if t < 1e-8
                break
            end
        end
        θ -= t * ∇Lθ
        ∇Lθ = ∇L(θ)
        iter += 1
    end
    return θ, ∇Lθ, iter
end

Regressão Logística¶

  • Vamos considerar agora que $y^i \in \{0, 1\}$;
  • Vamos usar o modelo $$ h_{\theta}(x) = \sigma(\theta^T\phi(x)) = \frac{1}{1 + e^{-\theta^T\phi(x)}}; $$
  • $\phi(x) = (1,x_1,\dots,x_p)^T$, tradicionalmente;
  • Num exemplo $x^i \in \mathbb{R}$, $$ h_{\theta}(x) = \frac{1}{1 + e^{-\theta_0 - \theta_1 x}}; $$
  • A função de perda que podemos usar é $$ \ell(p, y) = -\ln\Big[p^y(1 - p)^{1-y}\Big] = -y\ln p - (1 - y)\ln(1 - p). $$
